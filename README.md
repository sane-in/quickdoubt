# ⚡ QuickDoubt — AI Math Solver for Students

A web app where students type a math doubt, select their grade, and get a clear step-by-step solution powered by AI — instantly.

## 🌐 Live Demo
[Coming soon — deploying on Render]

## 🎯 Why I Built This
The idea came from a real student pain point — when you're stuck on a math problem at night, there's no teacher around. QuickDoubt gives instant, grade-appropriate explanations.

## ✨ Features
- Type any math question and get step-by-step solution
- Select your grade (3–12) for age-appropriate explanations
- Clean, simple UI built for students
- Powered by Llama 3.3 via Groq API

## 🛠️ Tech Stack
- **Backend:** Python, Flask
- **Frontend:** HTML, CSS, JavaScript
- **AI:** Groq API (Llama 3.3 70b)
- **Deployment:** Render
